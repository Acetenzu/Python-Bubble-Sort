# Python-Bubble-Sort
現在有一串混亂的符號（包含數字、英文大小寫字母），你必須將這些符號依 ASCII code value 由小到大排序，並且計算出排序過程中的交換次數。 

Input

⚫ 第 1 行：整數 n (1 ≤ n ≤ 100) 

⚫ 第 2 行：n 個字元 (數字 0-9、大寫字母 A-Z 或小寫字母 a-z)，以空白分隔。 

Output 

⚫ 第 1 行：排序後的字元序列，以空白分隔。 

⚫ 第 2 行：交換次數，格式：swaps=X *請使用 Bubble Sort 來排序*

Sample Input 1 

5

A 3 b a 0

Sample Output 1 

0 3 A a b 

swaps=6 

Sample Input 2 

4

z Z 9 a

Sample Output 2 

9 Z a z 

swaps=4 

Sample Input 3 

6

M a r i O 1

Sample Output 3

1 M O a i r 

swaps=9
